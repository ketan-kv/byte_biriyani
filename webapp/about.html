System Blueprint

How AMDAIS thinks like a domain analyst

The pipeline is fully autonomous but transparent, with editable controls and execution logs.

Pipeline stages

  1. Intent Agent: detects likely domain from full schema and sample values.
  2. Research Agent: builds domain KPI priors, thresholds, and vocabulary.
  3. Analysis Agent: profiles full dataset and builds descriptive, diagnostic, predictive layers.
  4. Insight Agent: produces actionable recommendations and executive storyline.
  5. Orchestrator: records logs and returns editable controls for reruns.

Editable controls currently exposed

  • Missing value strategy: none, mean, median, zero, drop.
  • Re-run from frontend with new controls and compare outcomes.
